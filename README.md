### Contributing
1. Clone this repo : `https://github.com/teambi0s/teambi0s.github.io.git`
2. CD in the repo's folder : `cd teambi0s`

4. In your browser navigate to : http://path/to/teambi0s/index.html
5. Enjoy your application.
